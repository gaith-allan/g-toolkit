# مسارات التعلم المتكاملة

هذا القسم يوفر خريطة طريق واضحة للمسارات التعليمية في المجالات الستة.

## 1. مسار مطور الويب (البرمجة)

**المرحلة 1: الأساسيات**
- **HTML/CSS:** إتقان بناء الهيكلية والتصميم.
- **JavaScript:** فهم الأساسيات، DOM Manipulation.
- **المصادر المقترحة:** [The Odin Project](https://www.theodinproject.com/)، [freeCodeCamp](https://www.freecodecamp.org/).

**المرحلة 2: الواجهة الأمامية (Frontend)**
- **React/Vue/Angular:** اختيار إطار عمل والتعمق فيه.
- **إدارة الحالة (State Management):** Redux أو Zustand.
- **المصادر المقترحة:** توثيق React الرسمي، كورسات على Coursera.

## 2. مسار مختبر الاختراق (الأمن السيبراني)

**المرحلة 1: الأساسيات**
- **الشبكات:** TCP/IP، OSI Model.
- **أنظمة التشغيل:** Linux (Kali Linux).
- **المصادر المقترحة:** [Hack The Box Academy](https://academy.hackthebox.com/)، [TryHackMe](https://tryhackme.com/).

**المرحلة 2: الاختراق المتقدم**
- **اختبار اختراق الويب:** OWASP Top 10، Burp Suite.
- **الهندسة العكسية (Reverse Engineering).**
- **المصادر المقترحة:** شهادات OSCP، دورات SANS.
