# قائمة التحقق الأمني الأساسية (Security Checklist)

هذه قائمة سريعة لضمان الحد الأدنى من الأمان لجهازك وحساباتك.

| الإجراء الأمني | التفاصيل | الحالة |
| :--- | :--- | :--- |
| **1. مدير كلمات المرور** | استخدم أداة مثل Bitwarden أو 1Password لإنشاء وتخزين كلمات مرور قوية وفريدة. | [ ] |
| **2. المصادقة الثنائية (2FA)** | تفعيل 2FA على جميع الحسابات الهامة (Gmail, GitHub, Social Media). يفضل استخدام تطبيقات مثل Authy بدلاً من الرسائل النصية. | [ ] |
| **3. تحديث النظام** | تأكد من أن نظام التشغيل والتطبيقات محدثة باستمرار لسد الثغرات الأمنية. | [ ] |
| **4. جدار الحماية (Firewall)** | تأكد من تفعيل جدار الحماية على جهازك (Windows Defender Firewall, UFW). | [ ] |
| **5. النسخ الاحتياطي** | قم بإجراء نسخ احتياطي منتظم لبياناتك الهامة (3-2-1 Rule). | [ ] |
